# Movies Landing Page 

A simple, responsive landing page for movies. Built using HTML, CSS, and JavaScript, this project showcases featured movies with a clean and modern design.  

## Features
- Responsive layout for desktop and mobile
- Hero section with featured movie
- Movie cards with images and titles
- Smooth scrolling navigation
- Modern design using CSS and animations
